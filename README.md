# StockAapl App

### A React Native App which showing apple live stock price using [iex](https://iexcloud.io/docs/api/) cloud

## Quick Start

`npm install`

`react-native run-android`

## Features:

> React Native app

> Using Real data 
 
> CandleStick module used

> Splash Screen
